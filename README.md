# Matilda Dash

Minecraft-style platformer game featuring Matilda the badger.

## Quick Start

**Requirements:** Docker

```bash
scripts\start.bat
```

Visit `http://localhost:8081`

## Controls

### Desktop
- **WASD** or **Arrow Keys**: Move
- **Space**: Jump

### Mobile
- **Touch Controls**: On-screen buttons for left, right, and jump

## Development

```bash
npm install
npm run lint
npm test
```

## Structure

- `src/` - Game source code
- `assets/` - Images, sounds, animations
- `scripts/` - Utility scripts
- `tests/` - Unit tests
